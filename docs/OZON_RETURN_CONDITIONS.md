# Условия возврата Ozon (декабрь 2025)

## Общие правила при возврате

При возврате товара Ozon:
1. **Возвращает продавцу:**
   - Вознаграждение за продажу (комиссию)
   - Плату за эквайринг

2. **Списывает:**
   - Сумму реализации товара (цена товара с НДС, с учётом скидок и акций)

3. **Остаётся неизменным:**
   - Начисления по услугам в рамках доставки до покупателя

---

## FBO (Fulfillment by Ozon)

### Расходы при возврате

| Услуга | Описание | Тариф |
|--------|----------|-------|
| Обратная логистика | На склад Ozon | Тариф FBO **без коэффициента** времени доставки |
| Обработка возврата | Приёмка в АПВЗ | **15 ₽** |

### Способы возврата:
- **На склад Ozon** — доставляем возврат, невыкуп, отмену на ближайший склад. Тариф = логистика FBO без коэффициента
- **Продавцу** — перевозим возврат в ваш регион. Тариф = логистика FBO в пункт выдачи

### Фиксация тарифов
> Фиксируем тарифы на обратную логистику товаров по схеме FBO на **60 дней** с даты создания заявки на поставку. После истечения срока — плата по текущим тарифам.

### Пример расчёта FBO

**Детали:**
- Категория: «Свечи и подсвечники», тип «Подсвечник»
- Цена: **1600 ₽**
- Эквайринг: **1,5%**
- Объём: **1 л**
- Среднее время доставки: **28 дней = 39 часов**
- Доставка до места выдачи: **≤25 ₽**

**1. Покупатель заказал товар:**
- Вознаграждение: 1600 × 40% = **640 ₽**
- Эквайринг: 1600 × 1,5% = **24 ₽**
- Логистика 1л = **46,77 ₽**
- Коэффициент: **1,480** (процент от цены: 2,40%)
- Итого логистика: 46,77 × 1,480 + 2,40% от 1600 = 69,22 + 38,4 = **107,62 ₽**
- Доставка до места выдачи: **10 ₽**
- **Продавец зарабатывает:** 1600 - 640 - 107,62 - 10 = **818,38 ₽**

**2. Покупатель сделал возврат:**
- Обратная логистика (без коэффициента): **46,77 ₽**
- Обработка возврата в АПВЗ: **15 ₽**

**Ozon возвращает:**
- Вознаграждение: **640 ₽**
- Эквайринг: **24 ₽**

**Итого удержим при возврате:** 818,38 - 997,77 = **-179,39 ₽** (убыток)

## FBS (Fulfillment by Seller)

### Расходы при возврате

| Услуга | Описание | Тариф |
|--------|----------|-------|
| Обратная логистика | На склад Ozon или продавцу | Тариф FBS **без коэффициента** времени доставки |
| Обработка возврата | Приёмка в АПВЗ | **15 ₽** |

### Способы возврата:
- **На склад Ozon** — тариф = логистика FBO без коэффициента
- **Продавцу** — тариф = логистика FBS без коэффициента

### Пример расчёта FBS

**Детали:**
- Цена: **800 ₽**
- Эквайринг: **1,5%**
- Объём: **1 л**
- Доставка до места выдачи: **≤25 ₽**
- Отгрузка: самостоятельно в СЦ

**1. Покупатель заказал товар:**
- Вознаграждение: 800 × 29,5% = **236 ₽**
- Эквайринг: 800 × 1,5% = **12 ₽**
- Обработка отправления: **20 ₽** (передача в сортировочный центр)
- Логистика 1л = **81,34 ₽**
- Доставка до места выдачи: **10 ₽**
- **Продавец зарабатывает:** 800 - 236 - 12 - 20 - 81,34 - 10 = **440,66 ₽**

**2. Покупатель сделал возврат:**
- Обратная логистика: **81,34 ₽**
- Обработка возврата в АПВЗ: **15 ₽**

**Ozon возвращает:**
- Вознаграждение: **236 ₽**
- Эквайринг: **12 ₽**

**Итого удержим при возврате:** 440,66 - 648,34 = **-207,68 ₽** (убыток)

## realFBS (Standard / Express)

### Расходы при возврате

Расходы зависят от **способа получения возвратов**:
- «Лёгкий возврат» Почтой России
- Возврат по вашей инструкции

### Пример расчёта realFBS

**Детали:**
- Цена: **800 ₽**
- Эквайринг: **1,5%**
- Объёмный вес: **1 кг**
- Доставка: из Казани в Москву
- Метод: партнёрами Ozon в пункт выдачи партнёра

**1. Покупатель заказал товар:**
- Вознаграждение: 800 × 27% = **216 ₽**
- Эквайринг: 800 × 1,5% = **12 ₽**
- Доставка партнёром Ozon: **292 ₽** (за 1 кг объёмного веса)
- Агентское вознаграждение: **20 ₽**
- **Продавец зарабатывает:** 800 - 216 - 12 - 292 - 20 = **260 ₽**

### Расходы на возвраты при доставке в ПВЗ Ozon

| Услуга | Партнёры Ozon | Своя или другая служба |
|--------|---------------|------------------------|
| Обработка возврата в ПВЗ | По тарифу «Обработка возвратов, отмен, невыкупов Партнёрами Ozon» | По тарифу «Обработка возвратов, отмен, невыкупов Партнёрами Ozon» |
| Обратная логистика (невыкуп/отмена без брака) | На склад Ozon — по тарифу FBO «Обратная логистика» | На склад Ozon — по тарифу FBO «Обратная логистика» |
| Обратная логистика (клиентские возвраты) | Вернём товар к вам — по тарифу «Лёгкого возврата» | Вернём товар к вам — по тарифу «Лёгкого возврата» |

## Формулы для расчёта

### Стоимость возврата FBO
```
return_cost_fbo = base_logistics_cost + 15 (обработка)
```

### Стоимость возврата FBS
```
return_cost_fbs = base_logistics_cost_fbs + 15 (обработка)
```

### Ожидаемая стоимость возвратов (с учётом % выкупа)
```
expected_return_cost = return_cost × (100 - redemption_rate) / 100
```

### Эффективная логистика за единицу (с учётом возвратов)
```
effective_logistics = logistics_cost + expected_return_cost
```

---

## Ключевые моменты

1. **Обратная логистика БЕЗ коэффициента** времени доставки (в отличие от прямой)
2. **Обработка возврата = 15 ₽** фиксировано для всех схем
3. **Ozon возвращает** комиссию и эквайринг при возврате
4. **Ozon НЕ возвращает** стоимость логистики доставки
5. **Фиксация тарифов FBO** на 60 дней с даты поставки

## Влияние % выкупа на юнит-экономику

При выкупе 80%:
- 80 из 100 заказов выкупают → платим только прямую логистику
- 20 из 100 возвращают → платим прямую + обратную логистику + обработку

**Формула эффективной логистики:**
```
effective_logistics = direct_logistics + (return_logistics + 15) × (1 - redemption_rate/100)
```
